# Expense Tracker App 💸

A Flutter-based mobile application to manage daily expenses efficiently.

---

## ✨ Features
- Add & Delete Expenses
- Category-based tracking
- Pie Chart Visualization 📊
- Dark Mode 🌙
- Local Storage using SharedPreferences
